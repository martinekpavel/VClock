# Valheim Clock
AI code this for time management over game UI. You can move it (left click), reset (left click on yellow part), setup (right click) and exit with middle mouse click.

You can download executable here:
[Release v1.0.0](https://github.com/martinekpavel/VClock/releases/tag/v1.0.0)
